# macOS 未签名安装说明

当前公开版本可能以未签名或未完成 Apple 公证的桌面安装包形式分发。

## 安装前说明

在 macOS 上，如果应用尚未完成签名或公证，系统可能会阻止直接打开。

## 常见处理方式

### 方法 1：通过“仍要打开”

1. 双击应用或 DMG 后，如果系统拦截，进入 **系统设置 > 隐私与安全性**
2. 找到关于该应用的安全提示
3. 选择 **仍要打开**

### 方法 2：右键打开

1. 在 Finder 中右键应用
2. 选择 **打开**
3. 在确认弹窗中再次点击 **打开**

## 风险提示

请仅从官方 GitHub Release 页面下载安装包，并确认版本号与发布说明一致。
