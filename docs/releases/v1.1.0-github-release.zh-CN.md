# AgentCore OS v1.1.0

AgentCore OS `v1.1.0` 正式发布。

这一版的重点不是继续堆更多角色和页面，而是把销售、客服和知识资产沉淀这三条高频闭环做得更稳。

## 本版重点

- 销售链路更完整：`Deal Desk -> Email Assistant -> Personal CRM -> Knowledge Vault`
- 客服链路更完整：`Support Copilot -> Reality Checker -> Knowledge Vault`
- 新增少量高价值数字员工角色白名单，而不是角色泛滥
- `Knowledge Vault` 进入流程资产层：可搜索、归档、编辑、复用、追溯
- 一键复用不再只是整段文本回填，而是结构化预填到销售和客服场景

## 新增能力

### 1. 数字员工白名单

当前版本引入了严格收敛的专家角色层：

- `sales_qualification_specialist`
- `outreach_draft_specialist`
- `support_reply_specialist`
- `reality_checker`
- `knowledge_asset_editor`

这些角色只绑定在具体流程节点中使用，强调稳定、准确、可审核，而不是无限增加 Agent 数量。

### 2. 销售和客服高频闭环

销售链路现在可以完成：

- 线索录入
- 资格判断
- 跟进草稿生成
- 人工审核
- CRM 收口
- 销售资产沉淀与复用

客服链路现在可以完成：

- 工单录入
- 建议回复生成
- `Reality Checker` 风险复核
- FAQ / 升级边界资产沉淀
- 知识资产复用

### 3. Knowledge Vault 流程资产层

`Knowledge Vault` 已支持：

- 流程资产搜索
- 启用中 / 已归档过滤
- 回到来源
- 标记已复用
- 一键复用
- 标题、适用场景、标签、正文编辑
- 查看来源轮次、资产键、创建时间、更新时间

## 安装方式

当前版本推荐安装方式：**命令行安装 / 从源码运行**

```bash
git clone https://github.com/aidi1723/agentcore-os.git
cd agentcore-os
npm install
npm run dev
```

推荐运行环境：

- Node.js `20+`
- npm

## 当前边界

当前版本仍然强调：

- 本地优先
- BYOK / API Key 驱动
- 人工审核边界清晰
- 资产沉淀优先于“全自动化炫技”

当前不建议宣传为：

- 完整无人公司
- 所有行业流程都已成熟覆盖
- 完全免人工审核
